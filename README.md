## 🎯 What It Does

Class schedules, attendance tracking, and daily planning in one focused workspace. Everything you need to stay on top of your academic life.

## ✨ Features at a Glance

| Feature | Benefit |
|---------|---------|
| 📅 Schedule management | Keep your timetable organized and always accessible |
| 🎯 Live class status | Know exactly which classes are pending, ongoing, or finished |
| 📊 Attendance tracking | Visual progress per course with custom goals |
| 🌓 Personalization | Dark/light themes and custom attendance targets |
| 📱 Works offline | Full functionality without internet |
| 🔒 Private & local | Your data stays completely on your device |

---

## 🚀 Core Features

| Feature | How It Works |
|---------|-------------|
| **Schedule Management** | Add, edit, or remove classes with exact timing details |
| **Live Status Updates** | Real-time display of pending, ongoing, and finished classes |
| **Attendance Tracking** | Mark attendance by date and view per-course percentages |
| **Light & Dark Themes** | Switch between color modes anytime |
| **Custom Goals** | Set your own minimum attendance target |
| **PWA Support** | Install on any device and use fully offline |
| **Private Storage** | Data stored locally with optional Firebase sync |
